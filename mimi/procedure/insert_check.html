Author: Mimi

Type of SQL Object: Procedure

Description: This procedure allows a user to enter a new quality check record into the database, including associating it with the correct batch, check type, and setting the initial status.

Justification: This is a vital procedure, since if users cannot enter new quality check records into the database, there is no point in using the database. This is better than a simple insert statement since it will validate the values before insertion. All users who are involved with the creation of batches will use it.

This code is hardened to SQL injections, try it out! Example: Try User ID = 2' or '1=1
