Author: Mimi

Type of SQL Object: Procedure

Description: For each product type, it allows users to the the ratio of passing and failing checks for all checks for that product type.

Justification: This will allow users to see which products are consistently failing and which might need further improvements. This will be useful when a company has a lot of products and need to see which are most problematic.

This code is hardened to SQL injections, there is no user input.

Expected Output: Shows the ratio for each product type.
