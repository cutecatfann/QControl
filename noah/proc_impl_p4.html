UpdateBatch -- Procedure

Author: Noah Mogensen

Description: Updates the production stage of the batch with the provided batch_id to the stage indicated by the provided stage_id

Justification: This allows for validation of the stage that the batch is being set to, to ensure that an incorrect stage of production is not set for a batch

To update the production stage of a batch, enter the numeric ID of the batch you would like to modify, and the desired stage id (also numeric):

Stages 1 and 2 are both valid stages for batch 1.
